## 🚀 快速启动

1.  **克隆仓库**：
    ```bash
    git clone https://github.com/TouHikari/shoppingWebPrac.git
    ```
2.  **进入项目目录**：
    ```bash
    cd shoppingWebPrac
    ```
3.  **打开 `index.html`**：
    直接在浏览器中打开 `index.html` 文件即可运行项目。
